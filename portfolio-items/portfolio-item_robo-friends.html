Robo-FriendsReact App

A Face-Recognition App

This super fun app was built with with React and provided me the opportunity to better understand the importance of using an API. The scope of this project was to implement ROBOHASH API and build an app around the API with contacts cards using React components and the Taychons CSS library. This Robo-Friends app uses a JSON file I created in my Github from my contacts, takes said JSON file, and uses the ROBOHASH API to present a picture of the contact.

This project was challenging and provided me an avenue to stretch my understanding of React component management, API management, and how to use a Github repo as a database to pull information into React. The demo uses the capability of Github Pages to run the app.

Somethings I would like to do in the future with this application would be to build a proper database that I could upload my contacts in to from my phone rather than uploading them to the repo. I could build a relational database with Node and PostgreSQL to better handle the incoming data.
