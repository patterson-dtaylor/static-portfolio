JavaScriptClock

A Real-Time Clock App

This project extended my understanding of working with the DOM, something I am continually practicing. Although it seems quite, simple, getting the time to line up correctly with the watch hands was difficult. Because of this, I enjoyed this project not only for its concepts, but for its challenges as well.

The styling of this project was simple enough, as well as configuring (mathematically that is) the time for the watch hands to tick. The challenge, as mentioned earlier, was rotating the clock hands in sync with the time function by using the DOM. A feature I would like to change in the app would the ability to change time zones or countries. I also think it would be great experience if this app could be turned into a desktop screen saver or Google Web background app.
