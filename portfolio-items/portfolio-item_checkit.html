CheckItA Check-List App

CheckIt

CheckIt is a todo iOS application where the user can create lists, items within those lists, check-off those items, and/or delete each list or item. After creating the list, the list(s) is then rendered in a random color. Each item a certain list is then rendered in darker variations of the color of the list.

Tools Used

  • ChameleonFrameworkA lightweight color Cocapod for iOS development.

    I used ChameleonFramework's ability to take TableView cells and render them in various colors. I then used ChameleonFramework's amazing ability to take a cell's color and render the cell's color variants to it's darkest color depending on the length of the list associated with the cell.

  • RealmA super fast mobile database alternative to CoreData.

    I used Realm to create a database to save the user's lists and list items. With Realm, users can create new lists and list items as well as deleting lists and its items. Although this database could have been created with CoreData, I learned how to use Realm to further my understanding of mobile databases.

  • SwipeCellKitAn easy to use Cocoapod to implement the swiping action used in the Mail app.

    One of the advantages of standing on the shoulders of giants is taking advantage of their amazing work and using their work to make your app better. Such is the case with my use of SwipeCellKit. Within CheckIt, I used the action provided by SwipeCellKit to give the user a familiar action used by Apple's Mail app. This UI component allows for a shorter learning curve due to familiarity of SwipeCellKit's actions.

Challenges and Further Improvements

This project presented some challenges with UITableViews and implementing Realm. However, documentation and research gave me the answers I needed to succeed with this project. As always, I think our projects are never perfect and always evolving. Some features I would like to develop to enhance the users experience would be a color customization feature when user adds a new list. Also, I would like to add a cloud database for users to log in, so when they change devices they could take their lists with them.
