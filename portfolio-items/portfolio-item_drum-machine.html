JavaScriptDrum Machine

A Drum App

The great thing about learning JavaScript is blending other passions with it to create something worthwhile. This drum machine project does just that. When I turned 15, I bought my first drum kit. I remember feeling how the sticks rebounded of the heads of the drums and ear shattering loud they could be. Now, using vanilla JS, HTML, and some CSS, I get to play around with the electronic skins while keeping my hearing intact.

Learning JS has been a mixed bag of fist pumping victories with smashing your head against the wall. It is a beautiful language that is extremely powerful which is why I try to use vanilla JS to develop most of my apps. Not that I dispute learning or using other libraries or frameworks, but I think it is better for development as a software developer.

As always, I think our projects are never perfect and always evolving. Some features I would like to develop to enhance the users experience would be a loop feature for saving a drum loop so the user could add to the loop. I would also like to add a metronome to keep steady time for the beat. Click below to check out the machine and get your inner Garth on!
