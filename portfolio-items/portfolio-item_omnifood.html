Omni-FoodWebsite

A Real-World Restaurant Profile

This is the final project of a web design course I took in January. I used this project to further explore my understanding of HTML concepts, CSS styling, and JS. To add to my understanding of JS, I also used a little JQuery to round out the edges of the UI experience. In the end, I think this project is a good representation of my aesthetic and developmental eye.

A few issues I had when working on this project was working with more complex ideas surrounding CSS styling. CSS is something that I continually practice because it is a weak point of mine. I also am not as familiar with JQuery as I am with JS, so there was a bit of a learning curve in the beginning. However, given a day or two (with Google by my side), I was able to grasp the library well enough to accomplish my task.
