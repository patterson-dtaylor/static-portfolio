Twittermenti

Twittermenti is a super fun app that searches for negative, positive, or neutral sentiments of Twitter handles and Twitter hashtags. Twittermenti then takes the search query and turns the findings, using the machine learning model, into emojis that emote the nature of the search.

Tools Used

  • Twitter Developer APITwitter's powerful API.

    Twittermenti uses Twitter's API to query a list of most recent 100 tweets when searching a hashtag or username.

  • SwifterA Twitter Framework written in Swift.

    Swifter is an embedded framework that works with the Twitter API all written in Swift. I used Swifter to access Twitter's API, run a query of the first 100 tweets based on the user's search field, and render data in the form of a JSON file. Swifter was super easy to use with the their amazing documentation.

  • SwiftyJSONA cocoapod that makes it easier to handel JSON data in swift.

    I used SwiftyJSON to render JSON data from Twitter's API and render said data to the controller. Although Swift can manage JSON data well with its JSON decoder Object, I wanted to branch out and learn another way to decode JSON data using a cocoapod.

  • Kaggle Sentiment140 DatasetA dataset based on Sanders' tweet sentiment dataset.

    I used the dataset as a part of the machine learning aspect of Twittermenti's ability to judge the sentiment of the first 100 tweets (+1 for positive, -1 for negative, and 0 for neutral). After calculating the judgement number, Twittermenti runs a simple switch/case block to render an emoji representing the current mood of the query.

Challenges and Further Improvements

Any project that requires new technology to learn comes with it challenges. This is why reading documentation is so important. Most of my challenges came from getting each technology to "talk" each other while maintaining MVC design structure and strong, clean object oriented programming. Some features I would like to develop to enhance the users experience would be a small UITableView to show a list of the user's recent queries and emojis associated with the query.
