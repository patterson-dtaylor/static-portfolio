Smart-BrainReact App

A Face-Recognition App

This is my final project of my Web-Developer course. This Smart-Brain app was built with with React to provide the user an awesome UI experience. The back-end was built with Node while the database was built with PostgreSQL. The scope of this project was to implement an open source API and build an app around the API. My Smart-Brain app can take the url of an image (with a human face) and recognize the face of the person through automation.

This project was challenging and provided me an avenue to stretch my understanding of component management, JSX and JavaScript functionality, and my overall comprehension of React. In addition, building the back-end was incredibly satisfying to see it come alive and register users on Heroku.
