What I Do

Front-end Development

I enjoy developing simple, clean client-side applications with HTML, CSS, API implementation, Vanilla JavaScript, and React. I have experience with using CSS frameworks such as Tachyons and Bootstrap to enhance the user's experience with the application's UI. Back-end Development

I have experience building server-side applications with Node.js, Express, Flask and Gunicorn. I also have experience building relational databases with PostgreSQL and used tools such as PostMan to fully develop back-end servers.

Version Control

My version control experience focuses mainly on using Git and Github to collaborate with others through open source projects. I achieve to write clean code, clear documentation, and keeping repositories organized with clear branch names.
